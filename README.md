# COVID-19 Genome Visualizer

## Run Locally

Clone the project

```bash
  git clone https://github.com/kereemyildiz/covid19-genome-visualizer.git
```

Go to the project directory

```bash
  cd covid19-genome-visualizer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
